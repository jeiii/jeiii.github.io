# 🚑 急救老司机的 HIS 备忘录



- 每次帮同事捋顺 HIS 系统的某个操作，转头自己再遇到相似问题，居然也会卡壳半天；刚解决完一个系统小故障，过阵子再碰上，又得从头琢磨一遍。

  这些重复又无效的功夫，实在磨人。

  索性把每次踩过的坑、理清楚的操作步骤、调试成功的小技巧，都随手记下来。

  不求写得多专业，只当是给自己和身边同事备一份 **随取随用的小手册** 。

  下次再忘，翻一翻就能捡起来，省点时间，多喘口气 —— 毕竟，咱还要留着力气握稳方向盘呢 .. ^_^ ..
